\# PramanaSystems Verifier CLI



Portable governance verification tooling.



This CLI allows independent verification of:



\- governance attestations

\- runtime manifests

\- execution evidence

\- compatibility guarantees

\- governance provenance



without requiring trust in the originating runtime environment.



\---



\# Goals



The verifier CLI is designed for:



\- auditors

\- compliance systems

\- regulated operators

\- runtime validation workflows

\- portable governance verification



\---



\# Planned Commands



\## Verify Runtime



```bash

pramanasystems-verifier verify-runtime

```



\## Verify Attestation



```bash

pramanasystems-verifier verify-attestation

```



\## Verify Release



```bash

pramanasystems-verifier verify-release

```



\## Verify Compatibility



```bash

pramanasystems-verifier verify-compatibility

```
