\# Attestation Verification Example



This example demonstrates independently verifiable governance attestations.



PramanaSystems allows governance decisions to be:



\- cryptographically signed

\- independently verified

\- portably audited

\- reproducibly validated



without requiring trust in the original runtime environment.



\---



\# Purpose



This workflow demonstrates:



1\. governance execution

2\. attestation generation

3\. independent verification



\---



\# Install



```bash

npm install

```



\---



\# Run



```bash

node verify-attestation.mjs

```

\# Guarantees Demonstrated



\- portable governance verification

\- independently reproducible trust

\- cryptographic attestation validation

\- deterministic governance evidence
