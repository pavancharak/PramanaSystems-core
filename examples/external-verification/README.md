\# External Verification Example



\## Purpose



This example demonstrates portable independent verification of pramanasystems governance artifacts outside the originating repository environment.



The example validates:



\- external SDK portability

\- independent package installation

\- portable verification semantics

\- release provenance verification

\- deterministic governance portability



This example intentionally avoids reliance on internal repository tooling.



\---



\# Verification Goals



The external verification flow demonstrates:



\- clean-room package installation

\- external SDK consumption

\- portable governance verification

\- independent runtime usage

\- reproducible governance semantics



Verification must remain executable independently of infrastructure ownership.
